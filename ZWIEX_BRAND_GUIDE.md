# ZwieX Brand Identity & Design System

## Brand Overview

**ZwieX** is a premium tech brand, creative studio, and web agency positioned at the intersection of innovation and excellence. The brand name combines "Zwie" (evoking Zwivhuya - good things, blessings) with "X" (representing cutting-edge technology), creating a memorable, globally-appealing identity.

**Tagline:** "good things meet innovation."

---

## Color Palette

### Primary Colors

- **Electric Cyan** (`#00D9FF` / `hsl(188 100% 50%)`)
  - Represents innovation, technology, and forward-thinking
  - Used for primary actions, accents, and interactive elements
  
- **Deep Space Black** (`#000000` / `hsl(0 0% 0%)`)
  - Primary background color
  - Creates a premium, futuristic foundation

- **Pure White** (`#FFFFFF`)
  - Accent color for contrast and clarity
  - Used for highlights and typography gradients

### Secondary Colors

- **Cyan 400** (`hsl(190 100% 60%)`)
  - Complementary cyan shade for gradients
  
- **Muted Grey** (`hsl(220 15% 65%)`)
  - Secondary text and subtle UI elements

---

## Typography

### Font System
- **Primary:** Sans-serif font family (system default)
- **Style:** Clean, bold, modern
- **Hierarchy:**
  - Hero: `text-6xl md:text-8xl` (72px-96px) - Bold, Extra Bold
  - Section Headers: `text-4xl md:text-6xl` (36px-60px) - Extra Bold
  - Body: `text-base md:text-lg` (16px-18px) - Regular weight
  - Captions: `text-sm` (14px) - Light weight

### Typographic Features
- **Text Gradients:** Primary cyan to white for brand name
- **Letter Spacing:** Tight (`tracking-tight`) for headlines
- **Font Weight:** Medium to Extra Bold for emphasis

## Design Elements

### Logo Design

The ZwieX logo consists of:
1. **Diagonal Band:** Vertical element rotated 12° with cyan glow
2. **Top Triangle:** Rounded triangle at top-left, outlined in primary cyan
3. **Bottom Triangle:** Rounded triangle at bottom-right, outlined in cyan-400
4. **Typography:** "ZwieX" in gradient (cyan to white)

**Logo Variations:**
- Full logo: 128px × 128px
- Compact header: 40px × 40px
- Favicon: 32px × 32px (recommended)

### Visual Effects

1. **Grid Pattern Background**
   - Subtle white grid (2% opacity)
   - 4rem spacing
   - Creates depth and structure

2. **Glow Effects**
   - `shadow-glow`: Standard glow on hover states
   - `shadow-glow-intense`: Intense glow for CTAs
   - Creates premium, tech-forward feel

3. **Glassmorphism**
   - `backdrop-blur-sm` for frosted glass effect
   - Used in cards, navigation, and overlays
   - Adds modern depth

4. **Gradient Orbs**
   - Animated pulsing orbs
   - Primary cyan at 20% opacity
   - Positioned strategically in backgrounds

### Animations

1. **Fade In** - `animate-fade-in`
   - Opacity + translateY on entry
   - 0.5s duration

2. **Scale In** - `animate-scale-in`
   - Scale from 95% to 100%
   - 0.4s duration

3. **Pulse Glow** - `animate-pulse-glow`
   - Brightness + opacity pulse
   - 3s infinite loop
   - Applied to logo and hero elements

4. **Slide Animations** - `animate-slide-in-left/right`
   - Slide from sides on entry
   - 0.6s duration

## Layout Principles

### Spacing System
- **Section Padding:** `py-20` to `py-24` (80px-96px)
- **Container Gap:** `gap-8` to `gap-12` (32px-48px)
- **Element Spacing:** `space-y-4` to `space-y-12`

### Grid System
- **Container:** Max-width with auto margins
- **Responsive:** 1 col (mobile) → 2 col (tablet) → 3 col (desktop)
- **Consistent gutters:** 32px spacing

### Visual Hierarchy

1. **Hero Section:**
   - Full viewport height
   - Centered content
   - Largest typography
   - Prominent CTA buttons

2. **Service Cards:**
   - Grid layout
   - Highlighted popular option
   - Clear pricing hierarchy
   - Feature lists with checkmarks

3. **Project Showcase:**
   - Infinite scroll carousel
   - Horizontal layout
   - Subtle brand badges

## Usage Guidelines

### DO ✅
- Use primary cyan for accents and CTAs
- Maintain high contrast ratios (WCAG AA)
- Apply subtle animations for engagement
- Keep typography large and readable
- Use glassmorphism sparingly for depth

### DON'T ❌
- Use primary cyan as large background areas
- Overload with glow effects
- Reduce font sizes below 14px
- Mix with bright, saturated colors
- Overcomplicate animations
